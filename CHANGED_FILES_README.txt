# Contrast Finance Backend v0.38.2

База: v0.38.1 / контрольная v0.38.0, где админка и кабинет менеджера зафиксированы как готовые.

## Что изменено в v0.38.2

Первая рабочая версия кабинетов глав департаментов Санжара и Рауфаля.

### Кабинет главдепа

Роль: `department_head`.

Кабинет полностью read-only:

- без создания мероприятий;
- без редактирования;
- без удаления;
- без действий по заявкам;
- без доступа к учредителям и общей калькуляции компании.

Вкладки:

- Обзор;
- Мероприятия;
- Заявки;
- Итог.

### Что видит руководитель отдела

Руководитель видит только свой отдел:

- план отдела;
- факт отдела;
- процент выполнения;
- менеджеров своего отдела и их прогресс;
- мероприятия своего отдела;
- заявки по видимым мероприятиям;
- текущую калькуляцию отдела;
- расходы, которые относятся к его отделу.

### Соавторские мероприятия

Если мероприятие в соавторстве между отделами:

- главдеп видит мероприятие полностью;
- в расчёт отдела попадает только доля менеджера/менеджеров его отдела;
- доля считается по `event_shares.share_percent`, как в админском обзоре и планах менеджеров.

### Итог отдела

Во вкладке «Итог» показывается актуальная текущая калькуляция:

- план;
- факт;
- выполнение;
- расходы;
- процент главы отдела 10% / 15%;
- предварительная ЗП главы отдела;
- остаток отдела после расходов и ЗП главы.

Учредители не показываются.
