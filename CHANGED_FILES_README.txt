# Contrast Finance Backend v0.37.26

Changed files only. Base: v0.37.25.

## Что исправлено

- Вкладка «Закрыть месяц»: фоновый пересчёт больше не стирает введённые, но ещё не отправленные данные в форме расхода.
- При обновлении калькуляции сохраняются значения полей «Название», «Сумма», «Распределение», ручные суммы Санжар/Рауфаль и «Комментарий».
- Сохраняется фокус и позиция курсора, чтобы можно было продолжать ввод без рывка экрана.
- Inline-редактирование суммы расхода тоже защищено от сброса при фоновом обновлении.

## Технически

- Добавлены `captureClosingExpenseDraft()` и `restoreClosingExpenseDraft()` в `app/web/app.js`.
- Перед перерисовкой вкладки форма расходов снимается в локальный snapshot, после перерисовки восстанавливается.
- Backend и миграции не менялись.

## Проверки

- `node --check app/web/app.js`
- `python3 -m compileall -q app`
