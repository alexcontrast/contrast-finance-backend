# Contrast Finance Backend v0.38.1 — changed files only

База: v0.38.0 stable checkpoint (кодовая база v0.37.26).

## Что исправлено

- Хотфикс кабинета менеджера: способ оплаты `Самозанятый` в смете больше не блокирует выпадающее меню сразу после выбора.
- `Самозанятый` теперь считается жёстко закреплённым только если по этой позиции уже есть активная заявка/оплата со способом `self_employed`.
- Если заявки по самозанятому нет, менеджер может обычным образом переключить способ оплаты обратно на `Налик`, `На карту` или `По счету`.
- При переключении с чернового `Самозанятый` на другой способ очищаются временные поля самозанятого: статус, вычеты и служебная заметка `Самозанятый: ...`.
