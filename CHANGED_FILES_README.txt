# Contrast Finance Backend v0.37.16 — plans tab and manager soft delete

## Что добавлено

Настроена вкладка админки «Задать планы».

- 12 полей плана компании по месяцам выбранного года.
- Проценты распределения отдела редактируются как проценты:
  - Санжар по умолчанию 66.67%;
  - Рауфаль по умолчанию 33.33%;
  - при изменении одного процента второй автоматически становится `100 - значение`.
- Единый процент личного плана менеджера от общего плана компании.
- Таблица сразу показывает рассчитанные суммы для Санжара, Рауфаля и личного плана менеджера по каждому месяцу.
- Кнопка «Сохранить планы» сохраняет все 12 месяцев через существующую таблицу `monthly_plans`.

## Удаление менеджера

На вкладке «Обзор» в шкалах планов менеджеров добавлена красная кнопка `×` справа от каждого менеджера.

Поведение:

- админ подтверждает удаление;
- менеджер переводится в `is_active = false`;
- кабинет менеджера сразу блокируется, потому что все API уже проверяют активность пользователя;
- мероприятия и заявки остаются в базе;
- факт отдела и компании за текущий месяц не пересчитывается назад из-за удаления менеджера.

Это soft delete без физического удаления строк из БД.

## Изменения в файлах

- `app/web/app.js`:
  - полноценный редактор планов на 12 месяцев;
  - сохранение планов за год;
  - кнопки удаления менеджеров в обзоре;
  - обработка soft delete менеджера.
- `app/web/styles.css`:
  - стили вкладки планов;
  - стили красной кнопки удаления менеджера.
- `app/api/routes/users_manage.py`:
  - `DELETE /users/{user_id}` для soft delete менеджера.
- `app/api/routes/admin_dashboard.py`:
  - расчёты месяца используют всех пользователей, включая отключённых, чтобы старые мероприятия не выпадали из факта отдела.
- `app/api/routes/monthly.py`:
  - сохранение и список планов закрыты ролью `admin`.
- `app/web/index.html`: cache-bust обновлён до `app.js?v=0.37.16` и `styles.css?v=0.37.16`.
- `app/core/config.py`: VERSION = `0.37.16`.
