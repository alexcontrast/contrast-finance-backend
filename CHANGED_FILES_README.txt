# Contrast Finance Backend v0.37.20

Changed files only.

## Что изменено

Точечная визуальная правка вкладки **«Задать планы»**:

- в колонках **Санжар**, **Рауфаль** и **Менеджер** выравнивание сделано по центру;
- колонка процента менеджера также центрируется, чтобы строка выглядела ровно;
- поля ввода процентов стали чуть ниже, с меньшим шрифтом и `box-sizing: border-box`, чтобы значения вроде `66.67` помещались целиком;
- процентные поля выравниваются по центру внутри своих маленьких окошек.

## Backend

Backend-логика и миграции не менялись.
