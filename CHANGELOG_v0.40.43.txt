# v0.40.43 — Telegram safe action rollback

- Added safe error handling for Telegram admin/manager inline actions.
- If a Telegram action fails after buttons are temporarily removed, the bot restores the clicked card from the current DB state.
- If automatic restore also fails, the bot leaves a visible error card with a "Вернуть кнопки" restore button instead of silently removing controls.
- Added logging for failed Telegram actions.
- Updated BOT_VERSION to CONTRAST_FINANCE_BOT_V0.40.43_SAFE_ACTION_ROLLBACK.
