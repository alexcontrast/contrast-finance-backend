# Contrast Finance 2.0 — v0.40.113

Точечная сборка для мобильного кабинета менеджера.

Причина: v0.40.112 визуально не применился в реальном менеджерском кабинете, потому что изменения были сделаны в основном CSS-слоем по старым общим селекторам. В v0.40.113 добавлены отдельные классы разметки и жёсткий role-атрибут `data-cf-role="manager"`, чтобы мобильные правила менеджера применялись именно к фактической странице менеджера.

Затронуты только:
- `app/web/app.js`
- `app/web/index.html`

Backend и Apps Script не менялись.
