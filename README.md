Contrast Finance Backend v0.35.0 — changed files only

Кабинет менеджера перестроен по логике старого сайта.

Что изменено:

- увеличен логотип Contrast
- логотип обрезан от лишних белых полей, без растягивания пропорций
- кабинет менеджера теперь без вкладок:
  - сверху данные менеджера и кнопка создания
  - ниже план месяца
  - слева список мероприятий с месяцем/годом и миникарточками
  - справа большая карточка выбранного мероприятия

Миникарточка мероприятия показывает:
- Бюджет = оборот
- Доход = доход компании
- заказчик
- дата
- способ оплаты/тип расчёта заказчика
- статус

Правая карточка мероприятия показывает:
- основные данные
- тип расчёта с заказчиком
- дата
- заказчик
- название
- агентская комиссия
- процент налогов
- переключатель Внешняя/Внутренняя смета как визуальный каркас
- внутренняя смета
- блок итогов

Создание мероприятия:
- кнопка + Создать мероприятие открывает модальное окно
- после создания мероприятие выбирается в списке

Добавление позиций:
- через кнопку + Добавить позицию внутри карточки мероприятия
- пока ручной налоговый статус для проверки калькулятора
- КГД/BIN подключим следующим шагом

index.html подключает app.js/styles.css с ?v=0.35.0

Миграций нет.
