- кнопка удалить позицию теперь есть и во внешней, и во внутренней смете

  - Менеджер
  - Налоги
  - НДС
  - Доход компании

  комиссия обычных позиций
  + агентская комиссия
  + НДС подрядчиков в зачёт
  + налоговые вычеты
  - налоги
- доход компании в онлайн-preview считается по правильной формуле:
  база менеджера
  - ЗП менеджера
  + координаторский остаток 50%
