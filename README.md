# Contrast Finance v0.5.26

Точечный frontend-патч для админской вкладки `Статистика`.

## Что проверить после деплоя
1. Открыть админку → `Статистика`.
2. В блоке `Доходы менеджеров` таблица должна быть не старой плоской, а группированной:
   - имя менеджера слева занимает высоту трёх строк;
   - справа от имени идут строки `Доход в компанию`, `ЗП`, `Координатор`;
   - дальше идут месяцы и `ИТОГО`.
3. Если вид не изменился, сделать hard refresh страницы, потому что изменены только `app.js` и `styles.css`.

## Changed-only
Для деплоя достаточно changed-only архива. Удалять файлы не нужно.
