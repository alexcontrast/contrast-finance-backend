   - способ оплаты нельзя менять

   - способ оплаты нельзя менять

   - способ оплаты нельзя менять

   - способ оплаты автоматически `Самозанятый`
   - способ оплаты нельзя менять
   - фамилия подтягивается из позиции

   - повторная заявка создаётся по уже закреплённой фамилии/статусу

   - появляется поле `Название позиции`
   - сумма заявки используется как сумма новой позиции
   - перед созданием заявки создаётся новая строка в смете
   - дальше работают те же правила: Нал / На карту / По счету / Самозанятый
