# Contrast Finance v0.5.31

Патч добавляет в веб-кабинет менеджера клиентскую PDF-смету по внешней смете.

## Что изменено

- Под внешней сметой появилась кнопка `Сформировать смету`.
- Кнопка доступна всегда, независимо от статуса мероприятия.
- PDF скачивается браузером и формируется из текущих данных внешней сметы.
- В PDF нет внутренних данных: факта, оплат, БИН/ИИН, КГД, НДС/вычетов подрядчиков, дохода и зарплат.
- Мобильная версия не менялась: кнопка скрыта через CSS.

## Проверка после деплоя

1. Открыть веб-кабинет менеджера на десктопе.
2. Открыть любое мероприятие.
3. Перейти на вкладку `Внешняя смета`.
4. Нажать `Сформировать смету`.
5. Проверить, что скачался PDF с клиентской сметой.
