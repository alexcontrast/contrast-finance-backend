Contrast Finance Backend v0.36.04 — changed files only

Админские действия с мероприятием и вкладка `Архив мероприятий`.

Добавлено:
1. В админской модалке мероприятия рядом с `Закрыть` появились кнопки:
   - Удалить
   - Принять
   - Деньги в кассе

2. Удаление:
   - доступно только для статусов `Черновик` и `На доработке`;
   - если есть активные оплаты, удалить нельзя;
   - правило продублировано на backend.

3. Принять:
   - доступно только для мероприятия `На проверке`;
   - переводит статус мероприятия в `Принято`.

4. Деньги в кассе:
   - переводит все неотменённые заявки мероприятия в статус `cash_received`;
   - переводит само мероприятие в статус `Деньги в кассе`;
   - после этого мероприятие попадает во вкладку `Архив мероприятий`.

5. Добавлена вкладка админки:
   - `Архив мероприятий`
   - там отображаются мероприятия со статусом `Деньги в кассе`.

Изменён backend:
- app/api/routes/events.py

Миграций нет.
