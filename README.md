  - налоги

  потому что он не добавляется в доходную базу

- Доход компании теперь считается так:
  комиссия обычных позиций
  + агентская комиссия
  + НДС подрядчиков в зачёт
  + налоговые вычеты
  - налоги
  - ЗП менеджера
  + координаторский остаток 50%
