- Прозрачный логотип без белой подложки.
- Кликабельная ссылка на сайт Contrast.
- Кликабельная кнопка Instagram.
- Комиссия агентства в итогах показывает сумму комиссии, а не общий subtotal.
- Убрано служебное примечание про внешнюю смету.
