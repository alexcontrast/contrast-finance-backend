# Contrast Finance v0.5.35

Патч поверх v0.5.34.

## Изменения
- В админке во вкладке «Закрыть месяц» расходы теперь могут быть отрицательными: сумма с минусом считается приходом и уменьшает расход месяца.
- В Telegram-боте быстрый расход также принимает отрицательную сумму, например `расход возврат -50000`.
- Распределение по отделам сохраняет знак суммы: отрицательный расход уменьшает расход выбранного отдела/месяца.
- Ноль по-прежнему не принимается как расход/приход.

## Деплой
Для деплоя достаточно changed-only. Удалять файлы не нужно.
