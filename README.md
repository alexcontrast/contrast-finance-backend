Точечный фронтовый патч кабинета менеджера.

- веб-блок личного плана стал зелёным;
- бейдж количества мероприятий расположен рядом с процентом;
- слово `мероприятие` склоняется по числу;
- в веб-карточке мероприятия `Удалить` стоит крайней правой;
- мобильный менеджер, админка, главдепы, backend и Telegram-бот не тронуты.

База: v0.40.119.
