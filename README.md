# Contrast Finance Backend v0.38.5

Полировка кабинета главдепа после v0.38.4.

## Что изменено

- В верхней шкале плана отдела значение `Цель/План` закреплено по центру, чтобы длинный `Факт` не сдвигал его вправо/влево.
- Таблица расходов главдепа больше не выводится снизу во вкладке `Итог`.
- Карточка `Расходы` во вкладке `Итог` стала кликабельной.
- По клику открывается отдельная независимая модалка расходов отдела:
  - отдельный `departmentExpensesModalBackdrop`, не `eventModalBackdrop` и не `plansModalBackdrop`;
  - компактная таблица без широких полей;
  - суммы выровнены по центру столбиков;
  - закрытие по кнопке `Закрыть` или клику по фону.
- Учредители и действия главдепу по-прежнему не показываются.

## Технически

- Изменены: `app/web/index.html`, `app/web/app.js`, `app/web/styles.css`, версия и cache-bust.
- Backend, БД и миграции не менялись.
- Проверки:
  - `node --check app/web/app.js`
  - `python3 -m compileall -q app`
