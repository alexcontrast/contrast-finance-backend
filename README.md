Contrast Finance Backend v0.31.8 — changed files only

Изменения в модалке мероприятия:

- верхние карточки стали компактнее
- карточки помещаются в один ряд на широком экране
- Доход компании выделен светло-зелёным
- Налоги теперь показывают процент налогового расхода:
  - ИП Contrast Event / ОУР без НДС: 12%
  - Упрощенка: 5%
  - Нал: 0%
  - если тип не распознан, процент считается из суммы налогов
- Менеджер 21% убран из верхних карточек
- Менеджер 21% добавлен в смету отдельной строкой-расходом
- в строке Менеджер 21% показывается начислено и оплачено, если backend отдаёт поле оплаты менеджера
- index.html подключает app.js/styles.css с ?v=0.31.8

Миграций нет.
