# Contrast Finance v0.5.28

Точечный патч вкладки админки `Статистика`.

## Что проверить после деплоя

1. Открыть админку → `Статистика`.
2. В таблице `Доходы менеджеров` проверить строку `План менеджера`: январь и остальные месяцы должны быть по центру.
3. Открыть админку на телефоне: таблицы статистики должны быть ровными, читаемыми и прокручиваться горизонтально.
4. При долгой первой загрузке смотреть Railway logs по строке:
   `PERF admin-year-statistics ...`

Для деплоя достаточно changed-only. Удалять файлы не нужно.
