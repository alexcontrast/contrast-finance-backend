# Contrast Finance v0.5.37

Точечный патч двусторонней live-синхронизации мероприятий между админом и менеджером.

Синхронизация использует существующее поле `updated_at`; новых полей и Alembic-миграций нет. Проверка выполняется раз в 5 секунд и при возврате во вкладку браузера. При наличии изменений перезагружаются данные текущего кабинета без F5.
