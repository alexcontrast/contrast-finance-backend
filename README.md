Contrast Finance Backend v0.35.77 — changed files only

Правка модалки `Мои оплаты`.

Изменено:
1. Оплаты группируются по позициям:
   - заявки по одной позиции теперь идут рядом
   - у каждой группы есть заголовок позиции
   - внутри группы компактные строки заявок

2. Соавтор:
   - в `Мои оплаты` теперь показываются все заявки текущего мероприятия
   - соавтор может отменить неоплаченную заявку по этому мероприятию,
     даже если её создал другой соавтор
   - оплаченные / деньги в кассе / отменённые заявки отменять нельзя

Backend не менялся относительно v0.35.76.
Миграций нет.
