Contrast Finance Backend v0.35.57 — changed files only

Фикс размера PIN-модалки и отображения email.

Точные причины:
1. PIN открывался через общий eventModalBackdrop, поэтому наследовал большую ширину обычной модалки мероприятия.
2. В auth.py уже передавался `email=user.email`, но в `AuthUserRead` поле email фактически отсутствовало.
   Поэтому Pydantic выкидывал email из ответа, и шапка не могла его показать.
3. `/app/bootstrap` также нужно явно отдавать email, чтобы он отображался после перезагрузки страницы.

Исправлено:
- добавлен `email` именно в `AuthUserRead`
- `/app/bootstrap` теперь отдаёт `email=user.email`
- после сохранения профиля frontend дополнительно сохраняет email из поля ввода, даже если старый backend ответит без email
- для смены PIN добавлен отдельный режим `pin-modal-mode`
- PIN-модалка стала узкой, вертикальной:
  - сверху старый PIN
  - снизу новый PIN
- обычные модалки мероприятий/профиля не затронуты
- app.js прошёл node --check
- Python-файлы компилируются

Миграций нет.
