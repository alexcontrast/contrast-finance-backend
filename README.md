Contrast Finance Backend v0.35.33 — changed files only

Быстрая правка миникарточек.

Frontend:
- зелёная обводка `is-open` теперь перемещается на реально открытую миникарточку через `state.selectedManagerEventId`
- при открытии мероприятия `selectedManagerEventId` обновляется сразу
- при клике по миникарточке обводка обновляется без ожидания полной перерисовки
- поля `Бюджет` и `Доход` в миникарточках стали зелёными
- при изменении сметы текущая миникарточка обновляет `Бюджет` и `Доход` в прямом эфире
- app.js проверен через `node --check`

index.html подключает app.js/styles.css с ?v=0.35.33

Миграций нет.
