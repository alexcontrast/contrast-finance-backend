# Contrast Finance Backend v0.37.15 — changed files only

## Что исправлено

Точечный фикс общих модалок после v0.37.14.

Проблема: кнопки админских действий (`На доработку`, `Принять`, `Деньги в кассе`) оставались в шапке модалки `eventModalBackdrop` и затем появлялись в других окнах: `Смена PIN` и `Данные менеджера`.

Причина: несколько разных окон используют один общий контейнер `eventModalBackdrop`, а старый блок `#eventModalActions` удалялся не при каждом открытии служебной модалки.

## Изменения

- `app/web/app.js`:
  - добавлен общий сброс `resetEventModalModes()`;
  - добавлено удаление старых админских кнопок `removeEventModalActions()`;
  - `Смена PIN`, `Данные менеджера`, `Оплатить`, `Мои оплаты` и обычное открытие мероприятия теперь очищают общий контейнер перед отрисовкой;
  - убран риск перетекания классов и кнопок между разными режимами одной модалки.
- `app/web/index.html`: cache-bust обновлён до `app.js?v=0.37.15` и `styles.css?v=0.37.15`.
- `app/core/config.py`: VERSION = `0.37.15`.

## Backend / DB

Backend-логика и миграции не менялись.

## Проверки

- `node --check app/web/app.js`
- `python3 -m compileall -q app`
