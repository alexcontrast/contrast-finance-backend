Contrast Finance Backend v0.35.96 — changed files only

Шлифовка админской модалки мероприятия.

Изменено:
1. В карточке `Менеджер 21%` убрана кнопка `Подать заявку`.

2. В таблицу сметы добавлены служебные строки надбавок, которые выставляются сверху заказчику:
   - Агентская комиссия
   - Налоги сверху для заказчика
   - НДС сверху для заказчика

3. Эти строки показываются внизу сметы и помогают увидеть,
   из чего складывается карточка `Оборот`:
   позиции сметы + агентская комиссия + налоги + НДС.

Логика сумм:
- Налоги берутся из summary.taxes_total или из internal_tax_amount + simplified_bank_tax_amount.
- НДС берётся из customer VAT полей: vat_to_pay / customer_vat_amount / vat_customer_amount.
- Агентская комиссия берётся из summary/event, если есть.
  Если отдельного поля нет, считается как остаток:
  оборот - сумма позиций сметы - налоги - НДС.

Backend не менялся относительно v0.35.95.
Миграций нет.

- app.js прошёл node --check
- Python-файлы компилируются
