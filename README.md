# Contrast Finance v0.5.23

Точечная правка админской вкладки `Статистика`.

## Что изменилось

- Убраны верхние KPI-миниокошки, потому что эти данные уже есть ниже в таблицах.
- `Динамика года` стала крупнее и читабельнее.
- Числа во всех таблицах статистики выровнены по центру.
- Менеджеры отсортированы по годовому доходу от большего к меньшему.
- Строки `Санжар доход` и `Рауфаль доход` выделены жирным.
