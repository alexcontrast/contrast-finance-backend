  - Вычеты
  - Комиссия
  - Оплачено

Логика:
- `Сохранить черновик` сохраняет видимые строки сметы
- `Отправить Саше` тоже сначала сохраняет строки, потом переводит статус в review
- `Самозанятый` автоматически показывает 10% вычетов
- `Налик` и `На карту` сбрасывают НДС/вычеты в 0
- `По счету` показывает кнопку КГД
- менеджеру разрешена КГД-проверка своих редактируемых позиций

Калькулятор:
- исправлен клиентский НДС: теперь используется VAT_RATE из config.py, сейчас это 16%
- внешняя смета хранится без НДС
- для ИП Contrast Event НДС добавляется сверху к клиентской сумме
- НДС к оплате = клиентский НДС - НДС подрядчиков в зачёт
