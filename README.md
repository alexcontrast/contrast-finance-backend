# Contrast Finance v0.40.123

Точечный фронтовый патч для веб-кабинетов менеджера и главдепа.

## Что изменено

- Менеджер / веб: цель и процент в окне личного плана теперь чёрные.
- Менеджер / веб: бейдж количества мероприятий затемнён до нормального системного зелёного.
- Главдеп / веб: окно главной цели стало ниже и компактнее — уменьшены шрифты, отступы и высота шкалы.

## Границы патча

Изменены только:

- `app/web/styles.css`
- `app/web/index.html`
- `CHANGELOG.md`
- `README.md`
- `app/CHANGED_FILES_README.txt`

Backend, Telegram-бот, админка и мобильные кабинеты не менялись.
