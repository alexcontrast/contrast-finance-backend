Contrast Finance Backend v0.35.47 — changed files only

Визуальная правка меню `Передать` / `Соавтор`.

Точная причина старого вида:
- действия менеджеров использовали общую модалку `plansModalBackdrop`
- у этой модалки в index.html статичная шапка `ПЛАНЫ / Задать планы`
- поэтому при любом использовании этой модалки сверху показывалось `Задать планы`

Исправлено:
- `Передать` и `Соавтор` больше не открывают большую модалку
- список менеджеров теперь выпадает компактным dropdown прямо из кнопки
- общая модалка планов больше не используется для этих действий
- `Задать планы` остаётся только для настоящей модалки планов
- добавлено закрытие dropdown по клику вне списка
- backend не трогался
- app.js прошёл node --check

Миграций нет.
