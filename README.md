# Contrast Finance v0.5.56

Точечный backend-патч распознавания НДС подрядчика при проверке БИН через КГД.

## Исправлено

- НДСная компания на ОУР больше не определяется как ОУР без НДС из-за формулировки поля `vatInfo`.
- Учитывается отдельная дата регистрации `vatDate` из официального API КГД.
- Явный статус снятия с НДС остаётся приоритетным.
- При непонятном статусе запускается резервная VAT-проверка вместо автоматического назначения «без НДС».

## Деплой

Миграций нет. После обычного деплоя у затронутой позиции нажмите карандаш возле БИН, затем повторно «Проверить»: НДС и вычеты будут пересчитаны и сохранены. Если по позиции уже есть активная заявка и карандаш заблокирован, повторную проверку может выполнить администратор.
