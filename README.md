Contrast Finance Backend v0.35.4 — changed files only

Правки кабинета менеджера и создания сметы:

Backend:
- EventCreate получил status с дефолтом draft
- POST /events создаёт мероприятие со статусом payload.status или draft
- при создании мероприятия автоматически создаётся позиция Координатор
- исправлено создание позиций: external_amount теперь считается как price * quantity * days
- позиции теперь реально добавляются, а не падают из-за payload.external_amount

Manager frontend:
- дата мероприятия показывается в формате ДД-ММ-ГГГГ
- в карточке мероприятия поле `Налоги %` убрано для обычных типов расчёта
- для Упрощенки показывается поле `Банк+налоги, %`
- в добавлении позиции убраны:
  - Тип
  - Факт
  - Способ оплаты
  - Налоговый статус
- при создании позиции задаётся только:
  - название позиции
  - сумма по смете
- Факт, способ оплаты, налоговый статус потом будут заполняться во внутренней смете
- из внутренней сметы убрана автоматическая строка `ЗП менеджера 21%`
- ЗП менеджера видна только в поле `Итого менеджеру`
- снизу карточки добавлены кнопки:
  - Сохранить черновик
  - Отправить Саше
- `Отправить Саше` переводит статус мероприятия в review

index.html подключает app.js/styles.css с ?v=0.35.4

Миграций нет.
