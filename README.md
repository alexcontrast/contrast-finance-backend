# Contrast Finance 2.0 v0.40.112

Патч только для мобильного кабинета менеджера.

## Установка
Заменить изменённые файлы из архива `CHANGED_ONLY` или развернуть полный архив версии.

## Проверка после установки
1. Зайти с телефона в кабинет менеджера.
2. Проверить шапку: логотип слева, роль справа, имя менеджера и кнопки компактно ниже.
3. Проверить строку месяц/год/обновить.
4. Проверить карточку личного плана.
5. Открыть миникарточку мероприятия и проверить прокрутку к основной карточке.
6. Проверить основную карточку: кнопки, поля, сметные вкладки и метрики должны помещаться без поломки сетки.

## Ограничение патча
Мобильная админка, мобильный главдеп и рабочая веб-версия не трогались.
