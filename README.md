Contrast Finance Backend v0.35.49 — changed files only

Правки бейджа соавтора.

Точные причины:
1. В миникарточке бейдж `Соавтор` рендерился отдельным элементом после бейджа статуса,
   поэтому он уходил на отдельную строку.
2. После удаления соавтора обновлялся dashboard/миникарточка, но `state.currentManagerEvent`
   и основная карточка не очищались от `is_coauthored/coauthor_name/share_percent`.
   Поэтому бейдж в основной карточке исчезал только после полного обновления страницы.

Исправлено:
- статус и `Соавтор` в миникарточке теперь находятся в одном `.mini-badge-row`
- добавлен `clearCoauthorStateForEvent(eventId)`
- после удаления соавтора локальное состояние очищается сразу
- после удаления соавтора основная карточка перерисовывается без бейджа
- app.js прошёл node --check

Backend не трогался.
Миграций нет.
