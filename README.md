Contrast Finance Backend v0.35.56 — changed files only

Правки профиля менеджера и смены PIN.

Точные причины:
1. `Данные менеджера` были добавлены как большая кнопка в manager-top-actions.
   Нужно было показывать эти данные в верхней шапке под большим `МЕНЕДЖЕР`.

2. В форме профиля был пункт `Без отдела`.
   Теперь его нет: доступны только отделы Санжар и Рауфаль.

3. `Сменить PIN` не открывал модалку.
   Кнопка просто показывала старую inline-карточку `changePinCard` внутри dashboardScreen,
   поэтому визуально “ничего не вылезало”. Теперь открывается нормальная модалка.

Исправлено:
- данные менеджера перенесены в верхнюю шапку под ролью
- рядом с данными добавлен маленький круглый карандашик
- большая кнопка `Данные менеджера` убрана из панели действий
- профильная модалка оставлена, но открывается из карандашика в шапке
- в выборе отдела нет `Без отдела`
- frontend показывает только отделы Санжар / Рауфаль
- backend тоже запрещает сохранить пустой отдел или отдел не Санжар/Рауфаль
- `Сменить PIN` теперь открывает модалку через eventModalBackdrop
- app.js прошёл node --check
- Python-файлы компилируются

Миграций нет.
