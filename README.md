# Contrast Finance v0.5.14

Hotfix для меню `Передать` / `Соавтор` в кабинете менеджера.

Изменены только фронтовые файлы:
- `app/web/app.js`
- `app/web/index.html`

После деплоя в консоли должно быть:

```text
Contrast Finance web app v0.5.14 loaded
```

Проверка:
1. Открыть кабинет менеджера.
2. Открыть карточку мероприятия.
3. Нажать `Передать`.
4. Меню должно открыться рядом с кнопкой и влезать в экран.
5. Нажать `Соавтор` — аналогично.
6. Проверить на мобильном: меню должно быть по ширине мобильного окна, без вылета вправо/вниз.
