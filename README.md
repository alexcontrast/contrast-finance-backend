Contrast Finance Backend v0.35.53 — changed files only

Улучшение бейджей миникарточек без точек.

Точная причина:
- в v0.35.51 ряды бейджей были зафиксированы, но для защиты от переноса использовались:
  `overflow: hidden`, `text-overflow: ellipsis`, `white-space: nowrap`
- из-за этого суммы бюджета/дохода и имя соавтора могли обрезаться через `...`

Исправлено:
- ellipsis для бейджей миникарточек заменён на JS-подгонку текста
- `fitMiniBadgeText()` уменьшает шрифт бейджа до минимального значения
- если шрифта всё равно мало, текст слегка сжимается по горизонтали через `scaleX`
- верхний ряд `Бюджет / Доход` остаётся фиксированным
- нижний ряд `Статус / Соавтор` остаётся фиксированным
- данные стараются помещаться полностью без `...`
- подгонка запускается после рендера dashboard, после live-обновления миникарточки и при resize окна
- app.js прошёл node --check

Backend не трогался.
Миграций нет.
