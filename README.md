Contrast Finance Backend v0.37.04 — changed files only

Фиксы заявок и money_status.

Изменено:
1. У отменённых заявок больше нет кнопки `Вернуть`.

2. Кнопка `Вернуть` в архиве заявок показывается только для заявок:
   - status = paid / Оплачено
   - money_status = cash_received / Деньги в кассе

3. Добавлен третий статус денег:
   - waiting_money / Ждём денег
   - cash_received / Деньги в кассе
   - cancelled / Отменено

4. У отменённых заявок статус денег теперь считается `Отменено`.
   Для новых отмен backend записывает money_status = cancelled.
   Для старых отмен фронт показывает money_status = cancelled автоматически.

5. В таблицах `Заявки` и `Архив заявок` восстановлен вывод:
   - Дата
   - Менеджер
   - Заказчик
   - Мероприятие

Миграций нет.
Новая колонка не добавлялась — используется уже созданное поле payment_requests.money_status.

Проверки:
- app.js прошёл node --check
- Python-файлы компилируются
