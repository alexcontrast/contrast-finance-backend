Contrast Finance Backend v0.35.48 — changed files only

Правки соавторства.

Точные причины:
1. Кнопка `Соавтор` не менялась на `Удалить соавтора`, потому что frontend не проверял `event.is_coauthored`.
2. Live-preview миникарточки считал 100%, потому что карточка открывалась через `/events/{id}`,
   а поля `is_coauthored/share_percent/coauthor_name` приходят из `manager-dashboard`.
   Поэтому до сохранения preview не знал про 50/50, а после сохранения dashboard уже возвращал 50%.
3. В карточке не было отдельного синего бейджа соавтора.

Исправлено:
- при открытии карточки данные `/events/{id}` дополняются данными из `manager-dashboard`
- live-preview применяет `share_percent` к:
  - `final_company_income`
  - `manager_salary`
- кнопка `Соавтор` на соавторском мероприятии превращается в `Удалить соавтора`
- `Удалить соавтора` вызывает `POST /events/{event_id}/coauthor/remove`
- кто нажал удаление, тому мероприятие полностью переходит по backend-логике
- добавлен синий бейдж `Соавтор: имя менеджера` рядом со статусом
- такой же бейдж добавлен в миникарточку
- app.js прошёл node --check

Backend не трогался.
Миграций нет.
