# Contrast Finance 2.0 — v0.40.116

Точечный патч мобильного кабинета менеджера поверх v0.40.115.

## Что изменено

- Уменьшены высота и шрифт кнопок/меню в мобильной шапке менеджера.
- Бейдж с именем менеджера сделан компактнее.
- В блоке личного плана цель и процент теперь в одной строке, количество мероприятий — справа.
- Панель полей карточки мероприятия сжата и разложена в 3 строки под мобильный экран.

## Границы патча

Изменены только:

- `app/web/app.js`
- `app/web/styles.css`
- `app/web/index.html`
- `CHANGELOG.md`
- `README.md`

Админка, кабинеты главдепов, backend и Telegram-бот не менялись.
