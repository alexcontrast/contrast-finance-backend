Contrast Finance Backend v0.35.97 — changed files only

Шлифовка служебных строк в смете админской модалки мероприятия.

Изменено:
- строки надбавок больше не выделяются отдельным левым зелёным бордером;
- убрана подпись `Сверху для заказчика`;
- названия строк стали короткими:
  - Комиссия
  - Налоги
  - НДС
- в колонке `Способ` теперь прочерк `—`, а не `Заказчику`;
- строка `Налоги` показывает только банковские/налоговые платежи по Упрощенке;
- если банковских/налоговых платежей нет, строка `Налоги` не показывается;
- агентская комиссия теперь показывается суммой, а не процентом:
  если в событии задан процент, сумма считается от суммы позиций сметы.

Backend не менялся относительно v0.35.96.
Миграций нет.

Проверки:
- app.js прошёл node --check
- Python-файлы компилируются
