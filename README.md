Contrast Finance Backend v0.35.79 — changed files only

Фикс отмены заявок соавтором.

Причина:
- frontend уже показывал заявки всего мероприятия соавтору
- но backend при отмене всё ещё проверял старое правило:
  менеджер может отменить только заявку, созданную им самим

Исправлено:
- backend теперь разрешает менеджеру/соавтору отменять любую неоплаченную заявку мероприятия,
  если у него есть право редактировать это мероприятие
- право редактирования уже проверяется через `require_payment_request_edit()`:
  владелец мероприятия или соавтор
- оплаченные заявки менеджер/соавтор по-прежнему отменить не может:
  - paid
  - cash_received
- админские права не менялись

Backend изменён:
- app/api/routes/payment_requests.py

Миграций нет.
