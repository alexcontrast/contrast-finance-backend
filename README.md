Contrast Finance Backend v0.36.07 — changed files only

Добавлена кнопка `Вернуть в работу` для принятого и архивного мероприятия.

Изменено:
1. У мероприятия со статусом `Принято` появляется кнопка `Вернуть в работу`.
   По нажатию статус мероприятия становится `На доработке`.

2. У мероприятия со статусом `Деньги в кассе` / в архиве появляется кнопка `Вернуть в работу`.
   По нажатию статус мероприятия становится `На доработке`.

3. При возврате архивного мероприятия в работу статусы оплат не меняются.
   Если оплаты уже были `Деньги в кассе`, они остаются `Деньги в кассе`.

4. После возврата мероприятие переносится обратно из `Архив мероприятий` во вкладку `Мероприятия`.

Backend изменён:
- app/api/routes/events.py
  - роут POST /events/{event_id}/revision теперь разрешает откат из статуса cash_received.

Миграций нет.

Проверки:
- app.js прошёл node --check
- Python-файлы компилируются
