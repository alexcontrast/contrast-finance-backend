Contrast Finance Backend v0.35.7 — changed files only

Исправления после проверки v0.35.6:

- Агентская комиссия теперь считается как процент, а не как сумма в тенге
- Агентская комиссия показывается отдельной обязательной графой под внешней сметой
- НДС считается от суммы: позиции + агентская комиссия
- Для ИП Contrast Event:
  позиции 1 000 000 + комиссия 15% = 1 150 000
  НДС 16% = 184 000
  Итого к оплате = 1 334 000
- Убрано пустое/лишнее окошко НДС с прочерком для типов без клиентского НДС
- Кнопка Сохранить черновик получила реальный обработчик после рендера карточки
- Кнопка Сохранить черновик сделана серой принудительно через отдельный класс
- Сохранение теперь привязано после отрисовки карточки мероприятия, а не до неё
