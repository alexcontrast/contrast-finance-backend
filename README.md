Contrast Finance Backend v0.37.09 — changed files only

Запрет удаления мероприятия, если деньги уже в кассе.

Изменено:
1. Backend:
   - DELETE /events/{event_id} теперь проверяет event.money_status.
   - Если money_status = cash_received, backend запрещает удаление:
     `Нельзя удалить мероприятие: деньги уже в кассе`.

2. Админка:
   - у черновика / на доработке кнопка `Удалить` остаётся видимой;
   - если деньги уже в кассе, кнопка становится неактивной;
   - подсказка: `Нельзя удалить: деньги уже в кассе`.

3. Менеджер:
   - кнопки удаления мероприятия также блокируются, если money_status = cash_received;
   - backend всё равно защищает удаление, даже если кнопку обойти.

Миграций нет.

Проверки:
- app.js прошёл node --check
- Python-файлы компилируются
