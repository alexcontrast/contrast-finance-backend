Contrast Finance Backend v0.35.59 — changed files only

Правка профильной модалки менеджера.

Точные причины:
1. Поле телефона в форме профиля брало raw `user.phone`, поэтому показывало `7700...`.
   В шапке уже использовался formatter `formatPhoneDisplay()`, а в форме — нет.
2. Профиль открывался через обычную широкую event-modal, поэтому форма была большой горизонтальной.

Исправлено:
- телефон в форме профиля теперь отображается как `+7700...`
- профильная модалка получила отдельный компактный режим `profile-modal-mode`
- профильная модалка стала узкой и вертикальной, как PIN:
  - имя
  - телефон
  - email
  - отдел
- обычные модалки мероприятий и PIN не затронуты
- app.js прошёл node --check

Backend не трогался.
Миграций нет.
