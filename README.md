Contrast Finance Backend v0.35.90 — changed files only

Фикс соавторства в админке.

Точная причина:
- кабинет менеджера уже считал мероприятия через event_shares;
- админка всё ещё считала каждое мероприятие как 100% у владельца;
- поэтому у менеджеров мероприятие делилось, а в админке нет.

Исправлено:
- admin-dashboard теперь использует event_shares так же, как manager-dashboard;
- если соавторства нет, мероприятие считается 100% у владельца;
- если есть event_shares, админка строит строки/итоги по долям;
- доход компании по отделам делится по менеджерам/отделам согласно долям;
- ЗП менеджера также делится согласно долям;
- общий факт компании не задваивается, потому что сумма долей даёт 100%.

Изменён backend:
- app/api/routes/admin_dashboard.py

Frontend не менялся, кроме bump cache index.html.
Миграций нет.
