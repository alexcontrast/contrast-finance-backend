# Contrast Finance v0.5.34

Точечный frontend-патч клиентской PDF-сметы в веб-кабинете менеджера.

## Что изменено

- В шапке PDF-сметы теперь две кликабельные кнопки в одном стиле: сайт и Instagram.
- Текст в шапке PDF увеличен.
- В таблицу добавлен столбец `Цена` после `Позиция`.
- Денежные значения в строках таблицы выровнены по центру.
