Contrast Finance Backend v0.36.00 — changed files only

Шлифовка админской модалки мероприятия и таблицы `Мероприятия`.

Изменено:
1. Верхняя карточка `НДС` переименована в `НДС к уплате`.

2. Карточка налогов теперь называется:
   `Налоги <процент> к уплате`.
   Для неё берутся внутренние налоги к уплате:
   taxes_total или internal_tax_amount + simplified_bank_tax_amount.

3. В смете модалки колонка `Способ` перенесена в конец:
   Позиция / Смета / Факт / Оплата / НДС / Вычеты / Способ.

4. В строках мероприятий всё старается помещаться в одну строку:
   - шапка таблицы без переносов;
   - статусы без переносов;
   - длинные текстовые поля обрезаются многоточием;
   - таблица получает горизонтальную прокрутку, если ширины не хватает.

5. В служебной строке сметы `Налоги` по-прежнему показываются только банковские/налоговые платежи по Упрощенке, если они есть.

Backend не менялся относительно v0.35.99.
Миграций нет.

- app.js прошёл node --check
- Python-файлы компилируются
