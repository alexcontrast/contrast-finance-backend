Contrast Finance Backend v0.35.92 — changed files only

Шлифовка вкладки админки `Мероприятия`.

Изменено:
1. Цвет строки мероприятия теперь зависит от отдела менеджера, а не от статуса.
   - Санжар: зелёная заливка
   - Рауфаль: синяя заливка
   - неизвестный/смешанный fallback: нейтральная заливка

2. Текст строк мероприятий сделан чёрным.

3. Мероприятие в соавторстве теперь отображается одной строкой.
   Менеджеры пишутся в одной ячейке через `/`.

4. Столбец `Менеджер` перенесён сразу после `Дата`.

5. Сортировка мероприятий в таблице — по дате от ранней к поздней.

Важно:
- backend v0.35.90 всё ещё отдаёт строки по долям для корректных итогов и обзора;
- на вкладке `Мероприятия` frontend группирует эти строки обратно в одну строку мероприятия.

Backend не менялся относительно v0.35.91.
Миграций нет.

- app.js прошёл node --check
- Python-файлы компилируются
