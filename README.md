Contrast Finance Backend v0.37.03 — changed files only

Фиксы после разделения статусов оплаты и денег.

Исправлено:
1. Бейдж `Статус денег` теперь такого же размера, как обычные бейджи статуса.

2. Админские кнопки мероприятия (`Вернуть в работу`, `Принять`, `Деньги в кассе`)
   больше не должны появляться в менеджерской модалке `Оплатить`.
   Действия мероприятия показываются только админу и только в модалке мероприятия.

3. Ряды кнопок выровнены:
   - кнопки действий заявки;
   - кнопки удаления/действий соавторов;
   - кнопки действий мероприятия.

4. Во вкладку `Заявки` возвращены кнопки:
   - `Оплачено`;
   - `Деньги в кассе`.

5. Во вкладку `Архив заявок` добавлена кнопка `Вернуть`.
   Она делает:
   - money_status = waiting_money / Ждём денег;
   - status = rejected / Отменено.
   При status=rejected backend также откатывает paid_amount, если заявка была оплачена.

Миграций нет.
Backend-структура не менялась относительно v0.37.02.

Проверки:
- app.js прошёл node --check
- Python-файлы компилируются
