# Contrast Finance Backend v0.38.6

Точечная правка шкал кабинета главдепа после v0.38.5.

## Что изменено

- Разобрана причина, почему `Цель/План` визуально оставался не по центру: предыдущая grid-раскладка делила строку как `1fr / 1fr / 72px`, поэтому второй столбец был центром не всей шкалы, а своей правой колонки.
- В верхней шкале отдела `Факт`, `Цель/План` и `%` теперь позиционируются независимо:
  - `Факт` строго слева;
  - `Цель/План` строго по центру всей шкалы;
  - `%` строго справа.
- Та же схема применена к строкам менеджеров отдела, чтобы длинный факт не сдвигал `План`.
- Модалка расходов из v0.38.5 не менялась.

## Технически

- Изменены: `app/web/index.html`, `app/web/styles.css`, версия и cache-bust.
- Backend, БД и миграции не менялись.
- Проверки:
  - `node --check app/web/app.js`
  - `python3 -m compileall -q app`
