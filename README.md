# Contrast Finance v0.5.27

Точечный патч вкладки админки `Статистика`.

## Что изменено

- Строка `Координатор` в таблице `Доходы менеджеров` теперь считается из строк сметы `Координатор`.
- Учитываются как новые typed coordinator rows, так и старые/перенесённые строки с названием `Координатор`.
- В статистику берётся реальная выплата координатору: 50% от суммы сметы.
- Числа в строках `ЗП` и `Координатор` выровнены по центру во всех месячных колонках.

## Деплой

Можно ставить `changed-only`. Удалять файлы и выполнять миграции не нужно.
