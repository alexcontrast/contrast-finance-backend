# Contrast Finance v0.5.55

Точечный патч дублей строки «Координатор» в сметах.

## Исправлено

- При деплое миграция автоматически оставляет в каждом мероприятии одну актуальную строку координатора, а старые дубли мягко скрывает.
- Если на дублях были заявки на оплату, они переносятся на сохранённую строку без потери истории.
- В базе добавлено правило: у мероприятия может быть только одна активная строка координатора.
- Повторный или параллельный запрос автосохранения теперь обновляет существующего координатора вместо создания новой строки.

## Деплой

Патч содержит backend и миграцию `0011_coord_singleton`. При обычном деплое `scripts/start.sh` применит её автоматически.
