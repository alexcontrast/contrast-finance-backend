# Contrast Finance 2.0 — v0.5.6

Чистовой патч после оптимизации вкладок админки.

## Что изменено

- Убран оставшийся шум `PERF web ...` из консоли браузера.
- Убран backend `PERF admin-dashboard-bundle ...` из Railway logs.
- Сохранены ускорения вкладок `Задать планы` и `Закрыть месяц` из v0.5.3/v0.5.4.

## Что не менялось

- Расчёты.
- Telegram-бот.
- Google Sheets экспорт.
- Кабинеты менеджера/главдепов/админки по интерфейсу.
