# Contrast Finance v0.5.24

Точечный патч вкладки админки `Статистика`.

## Что изменилось

1. Кнопка вкладки `Статистика`:
   - неактивная теперь тоже подсвечена зелёным;
   - активная остаётся зелёной как раньше.

2. Годовая статистика:
   - теперь учитывает ручной процент ЗП главдепа, заданный в `Закрыть месяц` через карандашик;
   - если по месяцу есть `sanzhar_head_percent_override` или `raufal_head_percent_override`, статистика использует его вместо авто-правила 10%/15%;
   - итоговая ЗП главдепов, расходы и чистый доход пересчитываются с учётом ручного процента.

3. Загрузка статистики:
   - при разовой ошибке 502/503/504 frontend автоматически делает один повторный запрос.
