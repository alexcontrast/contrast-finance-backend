Contrast Finance Backend v0.35.91 — changed files only

Шлифовка страницы админки `Обзор`.

Изменено:
- уменьшен размер шрифта имён менеджеров в карточках отделов;
- имя менеджера теперь старается помещаться в одну строку;
- если имя слишком длинное, оно аккуратно обрезается многоточием;
- рядом с именем менеджера добавлен маленький бейдж с количеством мероприятий за выбранный месяц;
- бейдж показывает только цифру, без текста;
- если у менеджера в выбранном месяце нет мероприятий, бейдж не показывается.

Подсчёт мероприятий:
- считается по строкам мероприятий текущего месяца;
- при соавторстве учитывается строка менеджера с его долей, как в текущей админской логике v0.35.90.

Backend не менялся относительно v0.35.90.
Миграций нет.

Проверки:
- app.js прошёл node --check
- Python-файлы компилируются
