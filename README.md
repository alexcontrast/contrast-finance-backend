# Contrast Finance v0.5.26

Патч для мобильной версии админской вкладки `Статистика`.

## Что изменилось

- Исправлены прозрачные/наезжающие sticky-столбцы в мобильных таблицах `Динамика года` и `Отделы`.
- Sticky-столбцы сделаны непрозрачными; скролл происходит справа от фиксированной зоны.
- В мобильной таблице `Менеджеры` сужены колонки имени и показателя, чтобы было видно больше месяцев.
- `Доход в компанию` сокращён до `Доход`.
- Год и кнопки в верхней панели статистики на мобильной версии выстроены в один ряд.

## Деплой

Достаточно changed-only. Удалять файлы не нужно.
