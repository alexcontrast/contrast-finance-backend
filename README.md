# Contrast Finance 2.0 — v0.5.36

Патч `v0.5.36 — Manager draft autosave` поверх v0.5.35.

## Что изменилось

В веб-кабинете менеджера добавлено тихое автосохранение черновика мероприятия и сметы:

- менеджер редактирует смету как раньше;
- после последнего изменения сайт ждёт короткую паузу и сам сохраняет черновик;
- после успешного автосохранения появляется короткая подсказка `Автосохранено`;
- если интернет пропал, показывается `Ждёт интернет`, а локальная копия остаётся в браузере;
- при возврате интернета сайт пробует отправить черновик на сервер;
- при закрытии вкладки последняя версия также сохраняется локально.

## Не менялось

- Backend API и модели БД;
- Alembic;
- Telegram-бот;
- заявки на оплату;
- PDF-сметы;
- Google Sheets export;
- админка;
- мобильная версия.

## Деплой

Для деплоя достаточно архива changed-only. Удалять файлы не нужно.
