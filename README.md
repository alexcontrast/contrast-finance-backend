Contrast Finance Backend v0.35.86 — changed files only

Глобальная мягкая загрузка для кнопок по всему сайту.

Что добавлено:
- общий слой soft-loading для всех кнопок
- если кнопка запускает async-действие, она показывает мягкую загрузку до завершения действия
- если действие короткое/синхронное, кнопка показывает короткий визуальный отклик
- кнопка приглушается и получает маленький spinner
- существующие явные загрузки через setButtonLoading(), например:
  - `Проверяем…`
  - `Сохраняем…`
  - `Отменяем…`
  - `Создаём…`
  остаются приоритетными и не ломаются

Исключения:
- disabled-кнопки не трогаются
- close-кнопки не получают мягкую загрузку
- кнопки с `data-no-soft-loading="true"` можно исключить вручную при необходимости

Backend не менялся относительно v0.35.85.
Миграций нет.

Проверки:
- app.js прошёл node --check
- Python-файлы компилируются
