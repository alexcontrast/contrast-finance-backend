Contrast Finance Backend v0.35.25 — changed files only

КГД / карточки / удаление.

Frontend:
- на кнопке КГД во время запроса показывается `…`, кнопка временно блокируется
- миникарточки со статусом `На проверке` выделяются зелёным
- открытая миникарточка выделяется отдельным синим акцентом, чтобы не путать со статусом
- мероприятие `На проверке` переводится в режим просмотра: поля/смета/КГД/удаление/сохранение недоступны
- сверху карточки добавлена красная кнопка `Удалить`
- кнопка удаления показывается только для Черновика и На доработке

Backend:
- добавлен DELETE `/events/{event_id}`
- мероприятие можно удалить только в статусах `draft` или `revision`
- мероприятие нельзя удалить, если по нему есть активные заявки на оплату
- позицию сметы нельзя удалить, если по ней есть активные заявки на оплату
- отменённые/отклонённые заявки не блокируют удаление

index.html подключает app.js/styles.css с ?v=0.35.25

Миграций нет.
