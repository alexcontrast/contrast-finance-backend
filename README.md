# Contrast Finance — v0.40.121

Точечный визуальный патч веб-версии кабинета менеджера.

## Что изменено

- Блок `Личный план` в веб-версии менеджера теперь использует фирменный зелёный цвет интерфейса, как основной фон страницы и кнопки `Обновить` / `Оплатить`.
- Бейдж количества мероприятий в блоке плана также переведён на фирменный зелёный градиент.
- Cache-bust обновлён до `0.40.121`.

## Границы патча

- Только фронт менеджерского кабинета.
- Backend, Telegram-бот, админка и главдепы не менялись.
