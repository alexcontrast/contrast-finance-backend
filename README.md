Contrast Finance Backend v0.35.82 — changed files only

Фикс переключения месяца/года в кабинете менеджера.

Причины:
1. Если в выбранном месяце нет мероприятий, старая открытая карточка могла оставаться справа.
   Из-за этого выглядело так, будто месяц не переключился.

2. Если для выбранного месяца ещё не задан план, backend менеджерского dashboard мог вернуть 404.
   Тогда старый экран оставался на месте.

Исправлено:
- при смене месяца/года сбрасывается выбранное мероприятие
- если в месяце нет мероприятий, список показывает:
  `Мероприятий за <месяц год> нет`
- правая карточка очищается и показывает:
  `Мероприятий за выбранный период нет`
- кабинет менеджера больше не падает, если план месяца ещё не задан:
  в этом случае план считается 0, но список мероприятий всё равно отдаётся
