# Contrast Finance v0.5.33

Патч исправляет внешний вид клиентской PDF-сметы в веб-кабинете менеджера.

## Что изменено
- Колонки PDF-сметы приведены к структуре сайта: `Позиция`, `Кол-во`, `Дни`, `Сумма`, `Примечание`.
- Поправлено выравнивание цифр в таблице.
- Добавлен route для прозрачного PNG-логотипа, чтобы в PDF не использовался JPG с белой подложкой.
- Кнопка Instagram стала компактнее и аккуратнее вписана в шапку.

## Деплой
Достаточно загрузить changed-only архив. Удалять файлы не нужно.
