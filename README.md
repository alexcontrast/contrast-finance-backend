# Contrast Finance v0.5.30

Точечный frontend-патч для кабинета менеджера.

## Что изменено

В сметах кабинета менеджера исправлена навигация стрелками внутри полей:

- стрелка влево/вправо двигает курсор по числу или тексту внутри текущего поля;
- переход на соседнее поле происходит только когда курсор уже дошёл до края;
- стрелки вверх/вниз и Enter продолжают работать как табличная навигация.

## Изменённые файлы

- `app/web/app.js`
- `app/web/index.html`
