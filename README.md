# Contrast Finance Backend v0.37.18

Точечная версия поверх v0.37.17.

## Что изменено

### Вкладка «Задать планы»

В v0.37.16–v0.37.17 проценты деления плана хранились в строках месяцев на backend, но во frontend редактировались через три общих поля на весь год. Поэтому изменение процента визуально применялось ко всем месяцам и при сохранении могло переписать прошлые месяцы.

В v0.37.18 проценты вынесены в каждую строку месяца:

- у каждого месяца отдельное поле «План компании»;
- у каждого месяца отдельные проценты Санжар / Рауфаль;
- Санжар и Рауфаль в рамках одной строки автоматически держат сумму 100%;
- у каждого месяца отдельный процент личного плана менеджера;
- расчёты сумм обновляются только в изменённой месячной строке/по её значениям;
- сохранение отправляет в `monthly_plans` snapshot каждого месяца с его собственными процентами.

Итог: можно поменять проценты только для конкретного месяца, не меняя прошлые месяцы.

## Backend

Backend-логика и миграции не менялись. Таблица `monthly_plans` уже содержит месячные поля:

- `company_plan_amount`
- `sanzhar_share_percent`
- `raufal_share_percent`
- `manager_personal_plan_percent`
