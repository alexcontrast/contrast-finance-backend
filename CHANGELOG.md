# CHANGELOG — v0.5.28

## Admin Statistics mobile/readability tuning

- Исправлено выравнивание строки `План менеджера`: месячные значения, включая январь, теперь принудительно выровнены по центру.
- Для мобильной версии вкладки `Статистика` таблицы переведены в читаемый горизонтальный скролл вместо попытки втиснуть всё в ширину экрана.
- На мобильном заданы минимальные ширины для таблиц `Динамика года`, `Отделы`, `Доходы менеджеров`, чтобы колонки оставались ровными и читаемыми.
- Добавлен точечный PERF-лог для загрузки годовой статистики: общий тайминг, тайминг каждого месяца, количество мероприятий и заявок.

Не менялось: БД/Alembic, Telegram-бот, заявки, Google Sheets export, менеджерский кабинет и закрытие месяца.
