# Contrast Finance v0.5.53

## Manager estimate state integrity

- Исправлена потеря заполненной строки при Enter и других структурных перерисовках сметы.
- Редактируемая смета теперь всегда перерисовывается из `managerDraftItemsByEventId`, а не из потенциально устаревшей ссылки `currentManagerItems`.
- Повторная синхронизация неизменённого БИН больше не сбрасывает успешную проверку КГД, НДС и вычеты.
- Повторная запись любого неизменившегося поля стала no-op и не помечает строку изменённой без причины.
- Исправлены связанные сценарии: Enter, добавление позиции, удаление другой строки, смена вкладки сметы, drag-and-drop и перерисовка после КГД.

База данных и backend API не изменялись.
