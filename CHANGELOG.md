# Contrast Finance — v0.5.37

## Live sync мероприятий без поля version

- Добавлен лёгкий endpoint `GET /events/changes`, который использует существующий `events.updated_at`.
- Админский и менеджерский кабинеты раз в 5 секунд проверяют изменения по выбранному месяцу.
- При изменении мероприятия обновляется текущий кабинет без F5 и без полной перезагрузки страницы.
- Обновляются карточки мероприятий, даже если они не открыты.
- Если изменённое мероприятие открыто у менеджера и локальных несохранённых правок нет, обновляется и его содержимое.
- Если менеджер сейчас редактирует/автосохраняет черновик, внешнее обновление не затирает его ввод.
- Если у админа открыта карточка без активного режима редактирования, она также обновляется.
- Миграции БД и поле `version` не добавлялись.
