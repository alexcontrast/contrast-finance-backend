# Contrast Finance — v0.5.45

## Live sync без полной перерисовки карточек

- Убрана полная перезагрузка всего кабинета при каждом событии live sync.
- Менеджерский список обновляет только изменившуюся карточку мероприятия.
- Открытая карточка менеджера обновляется точечно через DOM morph: сохраняются фокус, курсор и скролл; активное поле не перезаписывается.
- В админке и кабинете главдепа обновляются только изменившиеся строки мероприятий.
- Открытая read-only модалка админа/главдепа обновляет заголовок, метрики и строки сметы без пересборки всей модалки.
- Если админ или менеджер прямо сейчас редактирует мероприятие, внешний sync не затирает активный ввод.
- Период проверки остаётся 5 секунд, схема `updated_at` сохранена, миграций БД нет.
