# Changelog

## v0.5.42 — Admin paid request state colors

- В админской смете колонка «Оплата/Оплачено» теперь использует тот же источник и тот же принцип, что кабинет менеджера.
- Сумма всех активных заявок по позиции отображается серым, пока хотя бы одна заявка не оплачена.
- Когда оплачены все активные заявки по позиции, общая сумма отображается чёрным.
- Если после полной оплаты появляется новая неоплаченная заявка, общая сумма снова становится серой.
- Логика применяется и в обычной админской модалке мероприятия, и в режиме редактирования сметы администратором.
- Backend и БД не изменялись.
