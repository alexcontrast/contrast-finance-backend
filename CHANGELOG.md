# Changelog

## v0.5.43 — Stale deleted estimate items fix

- Найдена конкретная причина ошибки «Одна или несколько позиций не найдены» при отправке мероприятия Саше.
- В локальном черновике браузера оставались ID строк сметы, которые уже были удалены из базы ранее.
- Повторное удаление таких строк теперь идемпотентно: отсутствующие ID больше не блокируют сохранение всей сметы и отправку мероприятия на проверку.
- Проверка активных заявок на существующих позициях сохранена.
- Уведомление об ошибке отправки/сохранения теперь показывается 7 секунд.
- Миграций БД нет.
