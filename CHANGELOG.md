# v0.5.28 — Admin mobile statistics manager scroll fix

- Исправлен мобильный скролл таблицы `Менеджеры` во вкладке `Статистика`.
- Причина: из-за `rowspan=3` у имени менеджера во 2-й и 3-й строках ячейка `ЗП`/`Координатор` становилась первой колонкой, а январь — второй, поэтому CSS ошибочно фиксировал январь вместе с левыми полями.
- Теперь в каждой строке менеджера всегда есть две служебные левые ячейки: имя/пустой spacer и показатель. Sticky-логика снова стабильная: скроллятся только месяцы.
- Desktop, backend, БД/Alembic, Telegram-бот, заявки, менеджерский кабинет и Google Sheets export не менялись.
