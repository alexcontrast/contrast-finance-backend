# v0.5.31 — Admin statistics mobile sticky manager columns

## Изменено
- Убраны временные PERF-логи годовой статистики из backend endpoint `/google-sheets/year-statistics`.
- Убран браузерный `PERF web admin-year-statistics` из консоли.
- В мобильной версии вкладки `Статистика` таблица `Доходы менеджеров` стала читаемее при горизонтальном скролле.
- Длинные имена менеджеров теперь переносятся внутри фиксированной колонки и не перекрывают месячные данные.
- При горизонтальном скролле фиксируются две левые колонки: имя менеджера и показатель (`Доход в компанию` / `ЗП` / `Координатор`).

## Не менялось
- Расчёты статистики.
- Google Sheets export.
- БД/Alembic.
- Telegram-бот.
- Заявки, менеджерский кабинет и закрытие месяца.
