# CHANGELOG

## v0.40.67 — Auth boot no-flash

- Исправлен визуальный миг экрана входа при обновлении страницы, если на устройстве уже сохранён токен входа.
- Начальное состояние страницы теперь `boot-mode`: интерфейс скрыт до завершения проверки сохранённой авторизации.
- При валидном токене сразу показывается нужный кабинет; при невалидном или отсутствующем токене показывается экран входа.
- Обновлены cache-busting версии `styles.css` и `app.js` в `index.html`.
- Backend и Apps Script не менялись.
