# CHANGELOG

## v0.5.36 — Manager draft autosave

- В веб-кабинете менеджера добавлено автосохранение черновика мероприятия/сметы.
- При изменении полей мероприятия или строк сметы изменения сохраняются с задержкой после последнего ввода, без перерендера всей карточки и без долгой перезагрузки после каждой цифры.
- При закрытии вкладки/страницы последняя версия черновика дополнительно сохраняется локально в браузере.
- Если интернет пропал, локальная копия остаётся в браузере и подхватывается при повторном открытии мероприятия; при возвращении связи сайт пробует отправить черновик на сервер.
- Ручная кнопка `Сохранить черновик` остаётся и работает как раньше.
- Изменения касаются только веб-кабинета менеджера. Backend, БД/Alembic, Telegram-бот, заявки, PDF-сметы, Google Sheets export, админка и мобильная версия не менялись.
