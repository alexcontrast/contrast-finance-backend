# CHANGELOG

## v0.5.31 — Manager web client estimate PDF

- В веб-кабинете менеджера под внешней сметой добавлена кнопка `Сформировать смету`.
- Кнопка активна всегда, независимо от статуса мероприятия.
- PDF формируется на стороне браузера из текущей внешней сметы, включая несохранённые правки в открытой карточке.
- В PDF попадают только клиентские данные внешней сметы: позиции, цены, количество, дни, примечания, комиссия агентства, банковские/налоговые платежи для упрощёнки, НДС и итоговая сумма к оплате.
- Внутренние данные (`Факт`, способ оплаты подрядчика, БИН/ИИН, КГД, вычеты, оплачено, доход, ЗП менеджера) в PDF не выводятся.
- Оформление сделано в стиле старой клиентской сметы Contrast: шапка с логотипом, данными мероприятия, таблицей и итоговыми строками.
- Кнопка скрыта в мобильной версии менеджерского кабинета.
- Backend, БД/Alembic, Telegram-бот, заявки, Google Sheets export и админская статистика не менялись.
