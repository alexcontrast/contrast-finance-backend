# v0.5.27 — Admin manager coordinator statistics fix

- Во вкладке админки `Статистика` исправлен расчёт строки `Координатор` в таблице `Доходы менеджеров`.
- Теперь координатор подтягивается из строк сметы с `item_type=coordinator` или названием `Координатор`, включая legacy/imported rows.
- В статистику менеджера попадает не вся сумма строки координатора, а реальная выплата координатору: 50% от суммы сметы.
- Исправлено выравнивание чисел в строках `ЗП` и `Координатор`: значения по месяцам, включая январь, теперь по центру.
- Cache-bust обновлён до `0.5.27`.

Не менялось: БД/Alembic, Telegram-бот, заявки, Google Sheets export, менеджерский кабинет, закрытие месяца.
