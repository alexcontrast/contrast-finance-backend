# v0.5.26 — Admin manager statistics grouped layout fix

## Изменено
- Во вкладке админки `Статистика` таблица `Доходы менеджеров` теперь явно перестроена в группированный вид:
  - первый столбец — имя менеджера;
  - имя менеджера объединено по высоте на 3 строки;
  - внутри группы идут строки `Доход в компанию`, `ЗП`, `Координатор`;
  - месяцы и `ИТОГО` остаются справа.
- Добавлены отдельные CSS-правила для визуального разделения групп менеджеров, чтобы таблица точно не выглядела как старая плоская версия.
- Cache-bust обновлён до `0.5.26`.

## Не менялось
- Backend, БД/Alembic, Telegram-бот, заявки, Google Sheets export, менеджерский кабинет и расчёты закрытия месяца не менялись.
