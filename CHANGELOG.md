# v0.40.113 — Mobile Manager Real Rebuild

## Что изменено
- Пересобрана мобильная версия кабинета менеджера по реальной разметке, а не только поверх старых CSS-селекторов.
- Для роли менеджера добавлен `data-cf-role="manager"`, чтобы мобильные стили применялись жёстко и не зависели от старых классов режима.
- Добавлены отдельные классы v113 для менеджерского dashboard/layout/плана.
- Мобильная шапка менеджера приведена к формату админки/главдепа: логотип слева, роль справа, кнопки в ряд ниже.
- Мобильный блок месяца/года/обновления уплотнён в одну строку.
- Мобильная карточка личного плана менеджера перерисована: факт, цель/процент/бейдж мероприятий, шкала и остаток.
- Миникарточки и основная карточка менеджера уплотнены для мобильной работы.

## Не изменялось
- Мобильная админка.
- Мобильный кабинет главдепов.
- Десктопная/веб-версия.
- Backend.
- Apps Script.
