# v0.40.115 — Mobile manager width containment

- Мобильный кабинет менеджера: добавлен финальный width-containment override после v0.40.114.
- Карточка личного плана теперь сжимается в ширину телефона: уменьшены крупная сумма, строка цели, процент, бейдж мероприятий и подпись «Осталось».
- Миникарточки мероприятий больше не растягивают страницу: зафиксированы `width/max-width/min-width`, уменьшены шрифты, бейджи и пилюли ужаты внутри двух колонок.
- Карточка мероприятия ужата под мобильную ширину: заголовок переносится, кнопки/поля/итоговые метрики не должны создавать общий горизонтальный скролл.
- Горизонтальный скролл оставлен только внутри блока сметы `#managerEstimatePanel` / `.estimate-table-wrap`.
- Правка ограничена мобильным менеджером (`body.manager-mode` / `body[data-cf-role="manager"]`). Админка, главдепы, desktop и Telegram-бот не менялись.
