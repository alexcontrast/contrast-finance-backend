# Contrast Finance v0.5.27 — Admin mobile statistics nowrap polish

## Изменено
- Только мобильная версия админской вкладки `Статистика`.
- В таблице `Менеджеры` зафиксированы обе левые колонки: имя менеджера и показатель. Горизонтальный скролл должен двигать только месячные цифры.
- `ЗП менеджера` сокращено до `ЗП`.
- Отключены переносы слов и цифр в мобильных таблицах статистики.
- Сужены и подогнаны мобильные sticky-колонки `Менеджер` / `Показатель`, чтобы справа было видно больше месяцев.
- В `Отделах` заголовок `Показатель` и значения в первом столбце не переносятся.
- Для длинных подписей на мобильной версии используется уменьшенный шрифт/ellipsis вместо переноса по буквам.

## Не менялось
- Desktop-версия статистики.
- Backend и БД/Alembic.
- Telegram-бот.
- Заявки, менеджерский кабинет, Google Sheets export.
