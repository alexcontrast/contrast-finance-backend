# CHANGELOG

## v0.5.33 — Client estimate PDF layout fix

### Исправлено
- Клиентская PDF-смета в веб-кабинете менеджера теперь использует колонки как на сайте: `Позиция`, `Кол-во`, `Дни`, `Сумма`, `Примечание`.
- Исправлено выравнивание денежных значений в таблице: цифры больше не должны съезжать в соседние колонки.
- Исправлена причина белого фона логотипа в PDF: добавлен backend route для `/web/contrast-logo-transparent.png`, поэтому браузер больше не падает на старый JPG с белой подложкой.
- Кнопка Instagram в шапке PDF сделана визуально легче: без грубой чёрной заливки, с зелёной рамкой, аккуратной иконкой и выравниванием по шапке.

### Не менялось
- Backend-расчёты, БД/Alembic, Telegram-бот, заявки, Google Sheets export, админская статистика и мобильная версия.
