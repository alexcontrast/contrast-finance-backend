# CHANGELOG

## v0.5.24 — Admin statistics head salary override fix

- Во вкладке админки `Статистика` неактивная кнопка вкладки теперь тоже имеет зелёную подсветку; активное состояние осталось зелёным как раньше.
- Исправлено: годовая статистика теперь учитывает ручной процент ЗП главдепа, установленный в `Закрыть месяц` через карандашик.
- В годовой статистике `ЗП` отделов и строка расходов/чистого дохода теперь пересчитываются с учётом `sanzhar_head_percent_override` / `raufal_head_percent_override` по конкретному месяцу.
- Для разовых 502/503/504 при загрузке статистики добавлен один автоматический повтор запроса через короткую паузу.
- Backend, DB/Alembic, Telegram-бот, заявки, менеджерский кабинет и Google Sheets export отправка не менялись.

## v0.5.23 — Admin statistics table tuning

- Удалён верхний блок мини-окошек KPI во вкладке `Статистика`.
- Таблица `Динамика года` сделана крупнее.
- Цифры во всех таблицах статистики выровнены по центру.
- Менеджеры сортируются по годовому доходу от большего к меньшему.
- Строки `Санжар доход` и `Рауфаль доход` выделены жирным.
