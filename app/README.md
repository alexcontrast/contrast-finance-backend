# Contrast Finance Backend v0.37.25

Точечная версия после v0.37.24.

## Что изменено

- Во вкладке «Закрыть месяц» в строках расходов добавлена маленькая кнопка редактирования суммы.
- Редактируется только сумма расхода; название, комментарий и тип распределения не меняются.
- При сохранении новой суммы расход обновляется локально в списке, а калькуляция пересчитывается в фоне через уже существующий debounce.
- Для `default_split`, `100% Санжар` и `100% Рауфаль` суммы отделов пересчитываются автоматически.
- Для ручного распределения (`custom`) новая сумма пропорционально сохраняет старое соотношение Санжар/Рауфаль.
- В полях добавления расхода добавлена клавиатурная навигация: из названия `→` или `Enter` переводит фокус в сумму, из суммы `←` возвращает в название, `Enter` добавляет расход.
- Таблица расходов получила аккуратные кнопки `✎`, `✓`, `↺`, `×` и компактное поле редактирования суммы.
