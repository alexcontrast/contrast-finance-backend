# Contrast Finance Backend v0.38.8

Changed-only patch based on v0.38.7.

## Что изменилось

Точечная визуальная правка страницы входа.

- Верхняя шапка полностью скрывается на login-экране.
- На странице входа остаётся только карточка входа.
- Слоган «Делаем больше, чем просто сметы» уменьшен.
- Зелёная декоративная черта перед слоганом убрана.
- Слоган выровнен по центру.
- При переходе в кабинет шапка снова отображается как раньше.

## Что не менялось

- Backend-логика не менялась.
- База данных не менялась.
- Миграций нет.
