# Contrast Finance Backend v0.37.23

Точечная версия после v0.37.22.

## Что изменено

- Оптимизирована вкладка «Закрыть месяц» при добавлении/удалении расходов.
- Добавление расхода больше не запускает тяжёлый `loadDashboard()` и не перерисовывает всю админку.
- После сохранения расход сразу появляется в списке расходов локально.
- Калькуляция закрытия месяца пересчитывается отложенно в фоне через debounce.
- При удалении расхода строка сразу убирается из списка, а расчёт также обновляется в фоне.
- Кнопка «Пересчитать» сохранена для ручного явного пересчёта.

## Миграции

Новых миграций нет.
