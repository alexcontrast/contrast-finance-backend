# Contrast Finance Backend v0.37.22

Changed files only.

## Что изменено

Точечная правка вкладки **«Закрыть месяц»**:

- открытие вкладки больше не может сохранять/пересохранять закрытие месяца;
- вкладка при открытии только загружает расходы, текущий расчёт и статус месяца;
- месяц закрывается только явным нажатием кнопки **«Закрыть месяц»**;
- кнопка **«Пересчитать»** для незакрытого месяца только обновляет расчёт на экране;
- если месяц уже закрыт, **«Пересчитать»** отдельно спрашивает подтверждение и только после этого перезаписывает snapshot закрытия;
- текст вкладки уточнён: она хранит расходы и расчёт, а закрытие выполняется только кнопкой ниже;
- backend-логика, таблицы и миграции не менялись.
