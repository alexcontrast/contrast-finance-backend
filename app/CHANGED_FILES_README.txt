# Contrast Finance Backend v0.38.3

Точечная полировка read-only кабинетов глав департаментов после v0.38.2.

## Что изменилось

### Кабинет главдепа

- Убраны верхние дублирующие карточки `План / Факт / Выполнение / Расходы / ЗП`.
- Убран текст `Только просмотр` под заголовком кабинета.
- В бейдже пользователя для роли `department_head` теперь показывается только имя без подписи роли.
- В обзоре удалён дополнительный блок мини-метрик, чтобы не дублировать информацию ниже.

### Менеджеры отдела

- Количество мероприятий менеджера теперь показывается маленьким бейджем с цифрой.
- Если мероприятий нет, бейдж не показывается.
- В строке прогресса менеджера `Факт / План / %` закреплены по колонкам, чтобы длинная сумма факта не сдвигала `План` из центра.

### Вкладки «Мероприятия» и «Заявки»

- Добавлены фильтры главдепа:
  - по заказчику;
  - по менеджеру;
  - по статусу мероприятия / заявки.
- В кабинете главдепа архивные мероприятия и заявки показываются вместе с текущими. Разделение на архивы остаётся только у админа.

### Вкладка «Итог»

- Убрана карточка `Остаток отдела`.
- Подогнаны размеры карточек и шрифтов, чтобы значения помещались внутри рамок.

## Технически

- Изменены только frontend-файлы и версия.
- Backend, БД и миграции не менялись.
- Проверки:
  - `node --check app/web/app.js`
  - `python3 -m compileall -q app`
